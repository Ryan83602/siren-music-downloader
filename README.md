# siren-music-downloader
## 轻量Python脚本
### 使用方法：
1.下载siren-downloader.py


2.安装Python环境

3.在cmd,或者终端中运行pip install requests

4.cd到运行目录:语法:cd <目标目录>

#### eg:你的.py文件放在C:/msr/

cd C:/msr/

#### eg:你的.py文件放在/Volumes/upan/msr/

cd /Volumes/upan/msr/

5.运行

cmd或终端中运行python siren-downloader.py
